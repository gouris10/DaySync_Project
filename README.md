# DaySync – Task Management Web Application

## 📌 Project Overview

DaySync is a web-based task management application developed using Spring Boot, HTML, CSS, and JavaScript.
The application helps users organize their daily activities by creating, managing, and tracking tasks efficiently.

The goal of this project is to provide a simple and user-friendly platform where users can manage their daily work and improve productivity.

## Features

* User Registration and Login
* Task Creation and Management
* Dashboard for viewing tasks
* Organize tasks such as **Today, Inbox, and Upcoming**
* Simple and responsive user interface
* Backend API built using **Spring Boot**
